# Fonts Directory

This directory contains the Season Mix-TRIAL font files used in the portfolio website.

## Required Font Files

Please upload the following font files to this directory:

1. **SeasonMix-TRIAL-Regular.woff2** - Regular weight (400)
2. **SeasonMix-TRIAL-Regular.woff** - Regular weight (400) fallback
3. **SeasonMix-TRIAL-Medium.woff2** - Medium weight (500)
4. **SeasonMix-TRIAL-Medium.woff** - Medium weight (500) fallback
5. **SeasonMix-TRIAL-SemiBold.woff2** - SemiBold weight (600)
6. **SeasonMix-TRIAL-SemiBold.woff** - SemiBold weight (600) fallback

## Note

- Use `.woff2` format for better compression (primary)
- Use `.woff` format as fallback for older browsers
- File names must match exactly (case-sensitive)
- These fonts are referenced in `/styles/globals.css`
